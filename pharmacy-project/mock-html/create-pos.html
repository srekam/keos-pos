POS Device Setup

To set up a new POS device:

  1. Click "ADD POS" to register a new device
  2. Download and install the KeOS POS app on your device
  3. Enter the activation code provided during registration
  4. Configure device-specific settings (printer, cash drawer, etc.)
  5. Test all functions before going live
